# Therapy for Love App

## Introduction

Therapy for Love is a compassionate app designed to provide support and resources for individuals seeking relationship advice and emotional well-being.

## Technologies Used

- **Supabase**: For secure and scalable data storage and real-time database capabilities.
- **Vercel**: For seamless and fast deployment of the application.

## Features

- User authentication and profiles
- Personalized relationship advice
- Resource library with articles and videos
- Community support forum
